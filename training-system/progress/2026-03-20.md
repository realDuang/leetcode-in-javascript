## Core Takeaways

### DP vs BFS/DFS 判别规则

1. **"最少/最短" + 重复子问题 → DP 优于 BFS**
   - BFS 求最短路没错，但状态空间有大量重复时 DP 更高效
2. **回溯超时 → 检查是否有重叠子结构 → 改 DP**
   - Word Break: 回溯会重复计算相同后缀
3. **贪心 vs DP:**
   - 股票只买卖一次 → 贪心（状态只依赖一个累积量）
   - 股票多次/冷冻期/手续费 → DP（状态机）

### 统一模式："凑目标值"系列

| 题目 | "硬币面额" | "amount" | 转移方程 |
|------|-----------|----------|----------|
| 322. Coin Change | coins | amount | `dp[i] = min(dp[i-coin]+1)` |
| 279. Perfect Squares | 1²,2²,3²... | n | `dp[i] = min(dp[i-j²]+1)` |
| 139. Word Break | wordDict | s.length | `dp[i] = dp[i-word.len] (if match)` |
