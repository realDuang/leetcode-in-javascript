### Deep Dive: 0/1 vs 完全背包

Extended discussion on why loop order matters:

```
0/1背包:   for (j = target; j >= num; j--)  // 倒序：大j先算，读旧值，物品只用一次
完全背包:  for (j = coin; j <= target; j++)  // 正序：小j先算，大j读新值，物品可重复
```

Walked through complete example: `coins=[2,3], amount=6`
- 正序: dp[6]=2 (3+3, coin reuse ✅)
- 倒序: dp[6]=∞ (can't reuse, 2+3=5≠6)
