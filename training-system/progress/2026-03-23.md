## Core Takeaways

1. **Half-open `[left, right)`:** length = `right - left`, never +1
2. **validCount ±1 must use `===`:** exact match moment, not `<=` or `<`
3. **Fixed-length window:** use `r - l === len` as trigger, not satisfaction condition
4. **Variable discipline:** shrink removes `dropCh`, not `ch`
