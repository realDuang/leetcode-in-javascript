## Key Mistakes & Lessons

### 416 Partition Equal Subset Sum
- **误判原因:** "分成两组"→ 联想到双指针
- **正确识别:** "分两组和相等" → sum/2 → "能否凑出目标值" → 0/1背包 → DP
- **反例证明双指针不行:** [1,5,5,11] 排序后双指针得到 [1,5] vs [5,11]，无法跳着选

### 152 Maximum Product Subarray
- **遗漏点:** 负数×负数=正数，必须同时追踪 max 和 min
- **正确转移:** `dpMax[i] = max(nums[i], dpMax[i-1]*nums[i], dpMin[i-1]*nums[i])`
- **识别信号:** 连续子数组+乘积+有负数 → DP + max/min双追踪

### 221 Maximal Square (第3次犯错)
- **3/17:** 误判为 BFS
- **3/19:** 检查原始矩阵值而非 dp 值
- **3/25:** min/max 搞反
- **口诀:** 正方形=短板效应=三方向取 **min** + 1
- **dp定义:** dp[i][j] = 以(i,j)为右下角的最大正方形**边长**（不是面积！）

## Pattern Summary: "凑目标值"系列

| Problem | "Coins" | "Amount" | Question | DP Transition |
|---------|---------|----------|----------|---------------|
| 322 Coin Change | coins | amount | 最少几个 | `dp[j] = min(dp[j], dp[j-coin]+1)` |
| 279 Perfect Squares | 1²,2²... | n | 最少几个 | `dp[j] = min(dp[j], dp[j-k²]+1)` |
| 139 Word Break | wordDict | s.length | 能不能 | `dp[i] = dp[i-word.len] (if match)` |
| 416 Partition Equal | nums | sum/2 | 能不能 | `dp[j] = dp[j] \|\| dp[j-num]` |
| 494 Target Sum | nums | (sum+target)/2 | 有几种 | `dp[j] += dp[j-num]` |

### "+/-分组"转化公式
`P - N = target, P + N = sum → P = (sum + target) / 2 → 0/1背包`
