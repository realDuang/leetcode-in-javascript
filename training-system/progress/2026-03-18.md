## Key Mistakes

### 53. Maximum Subarray
- **错误:** 定义 `dp[i]` 为"前i个元素中最大和"，转移方程为"正数就加，负数就跳过"
- **问题:** 跳过负数破坏了连续性，变成了挑选所有正数求和
- **正确:** `dp[i]` = 以 `nums[i]` 结尾的最大连续子数组和
- **转移:** `dp[i] = max(dp[i-1] + nums[i], nums[i])`
- **答案:** `max(dp[...])` 而非 `dp[n-1]`

---

## Core Takeaways

1. **子数组/子序列问题 → `dp[i]` 定义为"以 i 结尾的 xxx" → 答案 = `max(dp[...])`**
2. **"选或不选"决策模型** (House Robber): `dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
3. **空间优化:** 1D DP 若只依赖前两个状态，可用两个变量替代数组 O(n) → O(1)
4. **LIS 双循环模式:** 外层 i，内层 j < i，`nums[j] < nums[i]` 时 `dp[i] = max(dp[i], dp[j]+1)`
