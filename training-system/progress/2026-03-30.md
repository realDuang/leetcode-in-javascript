## Key Learnings

### Quick Select = Partition + Binary Search Thinking
- 快排的 partition 确定 pivot 位置
- 只递归包含目标的一半 → O(n + n/2 + n/4 + ...) = O(2n) = O(n)
- 等比级数 1 + 1/2 + 1/4 + ... = 2

### Trie + DFS + Backtracking (Word Search II)
- 多个 words 有共同前缀 → Trie 避免重复搜索
- 网格搜索 → DFS + 回溯（标记/撤销）
- Trie 剪枝：前缀不存在立即终止

### BST LCA
- 从根往下走（不是往上）
- p,q 都小 → 左；都大 → 右；分叉 → 当前即 LCA
