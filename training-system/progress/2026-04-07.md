## Key Learnings

### Trie 核心模板
```
TrieNode: { children: Map<char, Node>, isEnd: boolean }
insert:     逐字符走，没路建路，最后标 isEnd=true
search:     逐字符走，走不通→false，走完查 isEnd
startsWith: 同上，不查 isEnd
通配符'.':  遍历所有子节点，any 匹配即可（DFS 分叉）
```

### 滑动窗口适用条件（再次强化）
- ✅ 适用：元素非负、窗口有单调性
- ❌ 不适用：有负数 → 用前缀和或 DP
