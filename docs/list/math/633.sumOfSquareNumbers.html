# [633] 平方数之和

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a^2 + b^2 = c。

示例1:

输入: 5

输出: True

解释: 1 * 1 + 2 * 2 = 5

示例2:

输入: 3

输出: False

var judgeSquareSum = function(c) {
  let lp = 0;
  let rp = Math.floor(Math.sqrt(c));
  while (lp <= rp) {
    const res = lp * lp + rp * rp;
    if (res === c) {
      return true;
    } else if (res > c) {
      rp--;
    } else {
      lp++;
    }
  }
  return false;
};
